Bitmap to Raw Converter for Amiga SWOS: V1.02
- All-in-1 creating and testing graphics for Amiga SWOS
- Written by starwindz
- Release date: 2011.03.18

Description:
- I am developing bitmap file to raw file converter for Amiga SWOS under Windows platform.
- By using this utility you do not have to use Amiga utility such as ArtPro and Deluxe paint under Amiga emulation or real Amiga machine.
- Just modify the template bitmap file and convert it under Windows, then packed raw file will be automatically generated.

Features:
* Graphic editing
- Pitch
- Goal net and ball
- Revolving ad-board
- Main title
- Menu title
- Coaching zone refree
- Team1~3 and Goal Keeper
- Many others: charset, disk22, loader00, loader01, menubg, menubg2, menus, olddisk1, soccer_s
- All-in-1 creating and testing graphics
